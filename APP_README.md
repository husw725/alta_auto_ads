# Auto Meta ADS - Streamlit 应用

## 🚀 运行方式

### 本地运行

```bash
cd skills/auto-meta-ads-skill
streamlit run app.py
```

### 打开浏览器

访问：`http://localhost:8501`

## 🎯 功能

1. **投流查询**
   - 输入剧集名和地区
   - 自动选择视频
   - 查看视频链接

2. **Campaign 创建**
   - 可选创建完整 Campaign
   - 配置目标国家、预算、CPI
   - 获取预览链接

3. **对话功能**
   - 智能助手回复
   - 查看使用帮助

## ⚙️ 配置

侧边栏可调整：
- 目标国家
- 每日预算
- 目标 CPI
